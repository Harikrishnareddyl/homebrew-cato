# homebrew-cato

Homebrew tap for [Cato](https://github.com/Harikrishnareddyl/cato) — universal AI agent safety layer.

## Install

```bash
brew tap harikrishnareddyl/cato
brew install cato
```

## Update

```bash
brew upgrade cato
```

## Usage

```bash
cato init              # create .cato.toml in your project
cato adapt claude-code # set up AI agent hooks
cato check .env        # test rules
```
